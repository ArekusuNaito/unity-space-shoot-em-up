# unity-space-shoot-em-up
A simple Shoot 'em up made in Unity

Features:

- Player
    - Movement
    - Shooting
    - Lives
- Enemies
    - Health
    - Move Speed
    - The second kind of enemy shoots
- UI
    - Player's Lives
    - Score
- EnemySpawner
    - Simple wave generation
